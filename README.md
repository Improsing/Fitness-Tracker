# Fitness Tracker
Программный модуль фитнес-трекера, который обрабатывает данные для трех видов тренировок: бега, ходьбы и плавания.
Этот модуль рассчитывает результаты тренировки и выводит информационное сообщение о результатах тренировки.
Разрабатывал для собственного развития.

- Версия языка go 1.20
- Скачать завимости можно командой "go mod download"
- Запуск простенький "go run main.go"
